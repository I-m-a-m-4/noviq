# **App Name**: Quran Circle

## Core Features:

- Secure User Authentication & Onboarding: Allows users to sign in via Email/Password or Google. Includes an onboarding flow to select Niyyah (intention), daily commitment duration, and preferred reminder time, with all preferences stored securely in Firestore. The UI is designed with a futuristic, dark aesthetic and subtle glassmorphism effects for a streamlined and secure user experience.
- Personalized Dashboard Display: A clean, minimalist dashboard, presented with a futuristic dark aesthetic and subtle glassmorphism, displaying the user's current streak, selected Niyyah, quick access to start a daily session, and an overview of 'Your Circle Activity' in real-time.
- Circle Formation & Management: Users can easily create or join small accountability circles (2-4 members max), with the interface elements styled to integrate seamlessly into the app's modern, dark theme and glassmorphism design, fostering gentle social encouragement and storing circle data in Firestore.
- Circle Activity Status Overview: Displays simple daily completion statuses for each circle member (e.g., ✅ Completed, ⏳ Not yet) within the circle overview, utilizing minimalist iconography and clear, dark-themed UI components to promote presence without comparison or pressure, all updated in Firestore.
- Guided Daily Quran Session (AI-Powered): A generative AI tool assists in selecting a personalized Quran verse (Arabic text and translation) from an external API, tailored to the user's Niyyah. The verse display and interactive elements will feature a modern, dark aesthetic with subtle glassmorphism, and users can listen to audio recitation or skip.
- Reflective Journaling: Provides a prompt ('What is Allah teaching you in this verse?') for users to write short personal reflections on the displayed Quran verse, using a dark, clean text input area that integrates with the app's overall futuristic design. Reflections are stored in Firestore and can be skipped.
- Session Completion & Progress Tracking: Marks the daily session as complete, updates the user's current streak, and reflects their completion status within their accountability circle. A dedicated 'Progress' page visually presents total days active, current streak, and number of reflections through minimalist charts and data visualizations, adhering to the app's dark, modern UI style.

## Style Guidelines:

- Primary background: A deep, near-black hue (#050505) forms the canvas, providing a striking contrast for UI elements. Accent colors: Vibrant amber and yellow tones (e.g., #FFC438, #FFEBB1, amber-500 from Tailwind palette) will be used for interactive elements, highlights, and calls to action. Text and border colors: A range of whites and grays (e.g., white/5, gray-300, gray-400) ensure readability and subtle separation. Glassmorphism elements will feature semi-transparent dark backgrounds (e.g., rgba(20, 21, 24, 0.95)) with a blur effect.
- Headings will utilize 'Inter' in large, bold sizes with a 'tracking-tighter' style to convey impact and modernity. Body text will also use 'Inter' for clear readability across various weights. Code snippets and technical indicators will employ 'JetBrains Mono', offering a distinct and legible monospace font.
- Utilize a distinct set of line-based, duotone icons in the 'solar' style (similar to those from iconify.design). These icons will have a modern, slightly geometric aesthetic, providing clear visual cues without cluttering the minimalist interface.
- The layout will be clean, dark-themed, and content-centric, utilizing max-width containers for optimal readability. Key interactive elements and display panels will incorporate extensive glassmorphism with backdrop-blur effects, providing depth and a futuristic feel. Subtle 3D transformations (e.g., rotateX, rotateY) will be applied to hero sections and prominent mockups to create a dynamic and engaging visual experience. All designs will be mobile-first and fully responsive.
- Animations will be smooth, subtle, and intentional. Elements will utilize a 'reveal-on-scroll' effect for a staggered, engaging entrance. Hover states will feature gentle transitions for interactive feedback. More prominent components, especially those with 3D transforms, will include dynamic transitions (e.g., scaling and rotation changes on hover) to enhance interactivity and visual appeal.
